# Raspberry Pi 3 ile Akıllı Park Sensörü

Bu proje, Raspberry Pi 3 ve ultrasonik mesafe sensörü kullanarak araçların park mesafesini ölçen ve kullanıcıyı uyaran bir akıllı park sistemi geliştirmeyi amaçlar.

## Özellikler
- Raspberry Pi 3 ile gerçek zamanlı mesafe ölçümü
- Ultrasonik sensör ile park mesafesi tespiti
- Buzzer/LED ile sesli ve görsel uyarı
- (Varsa) Web arayüzü veya ekran üzerinden gösterim

## Kullanılan Teknolojiler
- Python
- Raspberry Pi 3
- Ultrasonik sensör (HC-SR04 vb.)
- (Varsa) Flask / HTML / CSS

## Kurulum
1. Raspberry Pi üzerinde Python kurulumu yapın.
2. Bu repodaki kodları indirin veya klonlayın.
3. Gerekli Python paketlerini yükleyin.
4. Uygulamayı çalıştırın.
